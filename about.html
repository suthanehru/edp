About the Project

Project Objective

The objective of this project is to design a car security system that enables the user to locate the vehicle and disable the motor once notification is sent to their mobile devices via a cell modem

Device Components

The microcontroller, Particle Electron development board with STM32F205RGT6 microchip, is used as a watchdog timer

The device sends text messages to the user via the Adafruit FONA 3G cellular breakout board which has a built-in GPS module for Location tracking

A magnetic Hall Effect based sensor, US5881 is attached to the vehicle which particularly aids in vehicle speed detection and positioning

Who can benifit from this device?

The device is easily modifiable and can be used for various purposes. Its main purpose is to protect vehicles from theft, however, it is not limited to just that. The device is very useful to the Taxi Industry and public transport because it allows them to organize and control thir vehicles. This device could be modified in different format to accomodate the needs.
